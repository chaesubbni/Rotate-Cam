# Rotate-Cam

본 프로젝트는 자율주행 차량의 핵심 센서인 카메라가 비, 먼지 등 악천후로 인해 시야를 상실하는 문제를 해결하기 위해 개발되었습니다. 별도의 와이퍼 장치 없이, 카메라 렌즈 하우징을 고속 회전시켜 발생하는 원심력으로 렌즈 표면의 물방울과 이물질을 효과적으로 제거합니다.

또한, 이 프로젝트는 모터 제어, UART 통신을 할 때 데이터시트를 상세히 분석해 회로도 및 전압 타이밍을 고려하여 외부 라이브러리 없이 레지스터 레벨에서 구현하였습니다.
이를 통해 시스템 오버헤드를 최소화하고 제어 효율을 극대화했습니다.

## 모터 제어 시스템

처음에 고전력(24V) 시스템을 안정적으로 제어하기 위해 L298N 모터 드라이버를 활용한 PWM 속도 제어 및 방향 제어 로직을 구현하였습니다.

**하드웨어 드라이버 분석:** * 24V 고전압 구동을 위해 외부 전원과 MCU 로직 전원을 분리하여 설계하였습니다.
    * **전압 강하 분석:** H-브릿지 구조 내 트랜지스터로 인한 전압 강하를 고려하여 모터에 전달되는 유효 전력을 계산하였습니다.
    * **스위칭 타이밍 최적화:** * 데이터시트 분석을 통해 모터 드라이버 내부의 Turn-on/Turn-off 지연 시간($T_{delay}$)과 상승/하강 시간($T_{rise}, T_{fall}$)을 파악했습니다.
        * 고속 PWM(예: 20kHz) 사용 시 발생하는 스위칭 손실과 발열 문제를 해결하기 위해, 하드웨어 특성에 맞춰 **PWM 주파수를 1kHz로 최적화**하여 제어 안정성을 확보하였습니다.
* **제어 기능 구현 (라이브러리 미사용):**
    * **PWM 속도 제어:** 외부 라이브러리 없이 16비트 타이머 레지스터를 직접 조작하여 정밀한 듀티비(Duty Cycle)를 제어하였습니다.
    * **정역방향 제어:** GPIO 레지스터 제어를 통해 H-브릿지의 전류 흐름 방향을 전환하여 정회전/역회전을 구현하였습니다.
    * **안전 제동(Brake Mode):** 단순 단전(PWM 0)이 아닌, HIGH/HIGH(또는 LOW/LOW) 논리 조합을 통한 액티브 브레이크 로직을 적용하여 물리적 관성을 즉각 제어하였습니다.
