# Rotate-Cam

본 프로젝트는 자율주행 차량의 핵심 센서인 카메라가 비, 먼지 등 악천후로 인해 시야를 상실하는 문제를 해결하기 위해 개발되었습니다. 별도의 와이퍼 장치 없이, 카메라 렌즈 하우징을 고속 회전시켜 발생하는 원심력으로 렌즈 표면의 물방울과 이물질을 효과적으로 제거합니다.

또한, 본 프로젝트는 데이터시트를 상세히 분석하여 회로 구조와 전압 타이밍을 고려해 외부 라이브러리 없이 **레지스터 레벨에서 하드웨어 제어(PWM, UART)를 직접 구현**하였으며, 이를 통해 시스템 오버헤드를 최소화하고 제어 효율을 극대화했습니다.특히 모터 구동 시 발생하는 물리적 관성을 즉각적으로 제어하기 위해, H-브릿지 회로 내부에서 모터의 역기전력 루프를 형성하여 회전력을 상쇄시키는 급제동 로직을 적용하여 시스템의 안전성을 확보했습니다.

회로 및 전압 타이밍에 대해 자세히 알고 싶으시다면 제 블로그 게시물을 읽어보시면 자세히 알 수 있습니다.

## 모터 제어 시스템

모터를 안정적으로 제어하기 위해 L298N 모터 드라이버를 활용한 PWM 속도 제어 및 방향 제어 로직을 구현하였습니다.
또한, 안정적인 구동을 위해 외부 전원과 MCU 로직 전원을 분리하여 설계하였습니다. 

- 전압 강하 분석: - H-브릿지 구조 내 트랜지스터로 인한 전압 강하를 고려하여 모터에 전달되는 유효 전력을 계산하였습니다.
- 스위칭 타이밍 최적화: - 데이터시트 분석을 통해 모터 드라이버 내부의 Turn-on/Turn-off 지연 시간($T_{delay}$)과 상승/하강 시간($T_{rise}, T_{fall}$)을 파악하여, PWM 주파수를 1kHz로 최적화함으로써 발열을 억제하고 스위칭 손실을 최소화했습니다.

* **제어 기능 구현 (라이브러리 미사용):**
    * **PWM 속도 제어:** 외부 라이브러리 없이 16비트 타이머 레지스터를 직접 조작하여 정밀한 듀티비(Duty Cycle)를 제어하였습니다.
    * **정역방향 제어:** GPIO 레지스터 제어를 통해 H-브릿지의 전류 흐름 방향을 전환하여 정회전/역회전을 구현하였습니다.
    * **안전 제동(Brake Mode):** 단순 단전(PWM 0)이 아닌, HIGH/HIGH(또는 LOW/LOW) 논리 조합을 통해 모터의 역기전력을 활용하여 급제동 시켰습니다.

## 모터의 역기전력을 활용한 급제동 시스템

물리적 충돌 방지를 위해 초음파 센서로 실시간 거리를 측정하고 임계값 이내로 장애물 접근 시 즉시 모터의 역기전력을 활용하여 모터를 급제동 시키는 안전 로직을 구현했습니다.

- 단순 PWM을 0%로 제어해 모터의 관성으로 정지시키는 것이 아닌 H-브릿지 상태를 High, High로 강제 전환해 이 과정에서 발생하는 역기전력을 활용하여 모터의 회전을 물리적으로 빠르게 감쇠시켜 급정거를 구현하였습니다.

## 비전 기반 지능형 수분/이물질 감지 알고리즘

카메라 렌즈 표면의 물방울을 실시간으로 감지하여 능동적으로 클리닝(모터 회전) 시퀀스를 트리거하는 비전 처리 알고리즘을 구현했습니다. 

단순한 흐림 감지 방식을 넘어, 투명한 수분의 물리적 특성을 고려한 밀도 기반 윤곽선 필터링 기법을 독자적으로 설계 및 적용했습니다.

### 핵심 동작 원리 

1. **격자 분할 및 윤곽선 추출:** 입력 영상을 20x20 격자로 세밀하게 분할한 후, `Canny Edge Detection` 알고리즘을 통해 전체 영상의 윤곽선을 추출합니다.
2. **골디락스 존(Goldilocks Zone) 필터링:** 각 격자 내의 윤곽선 밀도를 계산하여 특정 임계값 범위를 필터링합니다.
   - **하늘/단순한 배경:** 윤곽선이 거의 없음 (밀도 하한치 미달) -> 배제
   - **복잡한 피사체(나뭇잎 등):** 윤곽선이 너무 빽빽함 (밀도 상한치 초과) -> 배제
   - **물방울:** 둥근 테두리로 인해 **'적당히 자잘한 밀도'**를 가짐 -> **수분으로 최종 판별**
3. **자동화 트리거 연동:** 전체 시야 대비 물방울이 감지된 구역의 비율이 20%를 초과할 경우, 즉시 모터 구동 신호(1)를 발생시켜 하드웨어 제어부로 전달합니다.

### 감지 결과 시각화 (Detection Results)

영상의 좌측은 렌즈에 수분이 맺힌 원본 시야이며, 우측은 알고리즘이 물방울의 엣지 밀도를 분석하여 감지된 구역에만 노란색 오버레이를 활성화한 결과입니다.
